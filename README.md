# DoorDash-Customer-Spending-Analysis
Analyzes customer spending behavior on the DoorDash platform
(Excel Data Analysis)

Project Overview

This project analyzes DoorDash customer order data to understand spending behavior, order patterns, and revenue distribution across different customer groups. The analysis focuses on identifying trends in order frequency, spending levels, and customer purchasing habits.

The analysis was conducted using Microsoft Excel, leveraging data cleaning techniques, pivot tables, and visual dashboards to transform raw transactional data into actionable insights.

Tools Used

Microsoft Excel

Pivot Tables

Data Cleaning

Data Visualization

Basic Statistical Analysis

Key Analytical Questions

How is customer spending distributed across different orders?

What patterns exist in order frequency and spending behavior?

Which customers contribute most to total revenue?

How can spending patterns help businesses improve marketing strategies?

Key Insights

A small percentage of customers contribute a large share of total revenue, showing a typical Pareto (80/20) pattern in customer spending.

Customer order frequency varies widely, indicating different segments of casual and high-value customers.

Analyzing spending behavior helps identify high-value customers that businesses should prioritize for retention and targeted promotions.
